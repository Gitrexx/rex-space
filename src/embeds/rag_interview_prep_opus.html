Retrieval-Augmented Generation

MCQ 0/0
The mental model that survives every interview

RAG fails in exactly two places. Everything else is detail.

A RAG system retrieves context, augments the prompt with it, then generates an answer. That decouples knowledge (an index you can update) from reasoning (the frozen LLM).

So every failure is either a retrieval failure (the right information never reached the model) or a generation failure (the model had it and still got it wrong). Senior candidates diagnose which one — and measure them separately. Hold this 2×2 in your head through all ten modules.

Retrieval failureRight chunk absent from top-k. Fix: chunking, embeddings, hybrid, rerank, query transforms. Measure: context recall.
Generation failureRight chunk present, answer wrong. Fix: prompt, model, grounding. Measure: faithfulness.
